# Shared /database/ Worship Directory

This directory acts as the local storage folder for all worship karaoke tracks:
1. **Audio files**: Save your physical `.mp3` or `.wav` instrumentals inside this folder (e.g., `amazing_grace.mp3`).
2. **Lyric profiles**: You can save downloaded timing `.json` configurations here or use the App's built-in creation wizard.

The system is configured to resolve local assets instantly from this database scope relative to your app's root.
